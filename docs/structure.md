# Repository Structure

This repository combines a publishable technical article with a runnable companion lab.

- `ARTICLE.md`: article draft and source list.
- `README.md`: quickstart for the companion lab.
- `src/`: implementation code aligned with the article topic.
- `tests/`: unit tests for the companion lab.
- `scripts/validate_repo.py`: public repository validator.
- `assets/`: diagram references and image placeholders.
