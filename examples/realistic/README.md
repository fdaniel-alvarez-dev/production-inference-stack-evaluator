# Realistic Benchmark Inputs

Use this folder for benchmark metrics gathered from your own environment.

Do not publish raw production prompts, customer data, private traces, or infrastructure identifiers.

Minimum metadata to keep with any realistic metrics file:

- model name and revision
- framework version
- GPU model/count/memory
- driver and CUDA versions
- container image digest
- sampling parameters
- prompt set description
- warmup procedure
- concurrency sweep
- retry and failure handling
